# Dynamic Quote Generator

**Author:** Asenathi Sotshintshi

## Overview
A web app demonstrating **advanced DOM manipulation**, **web storage (localStorage)**, **category filtering**, and dynamic form creation using vanilla JavaScript.

## Features
✅ Display random quotes from different categories  
✅ Add new quotes dynamically  
✅ Persist quotes to `localStorage`  
✅ Filter quotes by category  
✅ Smooth fade-in/fade-out animations  
✅ Dynamic form creation without innerHTML

## How to run
1. Clone your repo or open the `dom-manipulation` folder.
2. Open `index.html` in your browser.
3. Test adding quotes, filtering by category, and viewing random quotes.
